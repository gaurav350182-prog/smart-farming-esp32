# Smart Electronic System for Farming 🌱

ESP32-based automated irrigation system — Final Year B.E. Project  
SSBT CoET, Bambhori, Jalgaon (KBC NMU) — 2025-26

## Features
- ✅ Automatic pump control based on soil moisture
- ✅ Rain detection — stops irrigation during rainfall
- ✅ Water level safety cut-off
- ✅ MQTT IoT monitoring over Wi-Fi
- ✅ RTC-based scheduling
- ✅ Manual override support
- ✅ LCD real-time display

## Hardware
ESP32 · DHT11 · Soil Moisture YL-69 · Rain Sensor YL-83  
Water Level Sensor · LDR · Relay Module · RTC DS3231 · LM2596S Buck Converter
